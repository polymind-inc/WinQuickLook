Why WinQuickLook?

A fast, lightweight file previewer that just works.

Lightning Fast

Instant file previews without waiting for heavy applications to launch. Just press Space and see your file immediately.

Wide Format Support

Preview images, documents, videos, audio files, Markdown, HTML, and many more formats out of the box.

Simple Keyboard Shortcut

Press Space to preview, Esc to close. That's all you need to remember.

Lightweight

Minimal resource usage. WinQuickLook runs quietly in the background, ready whenever you need it.

Modern UI

Clean, intuitive interface that fits seamlessly with the Windows look and feel.

Seamless Integration

Works directly with Windows File Explorer. No configuration needed—install and start previewing.

How It Works

Get started in seconds.

1

Install

Download from the Microsoft Store or install via WinGet.

2

Select a File

Browse to any file in Windows File Explorer and click on it.

3

Press Space

Instantly preview the file. Press Esc to close when you're done.
